How to insert JS into a HTML:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<script scr="app.js"></script>
<script>console.log('Direct code');</script>
</body>
</html>
```
